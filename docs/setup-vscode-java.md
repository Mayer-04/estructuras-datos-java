# Configuración de Visual Studio Code para Java

En VS Code, trabajar con paquetes en Java es más manual que en NetBeans.

## Extensiones recomendadas

- **Extension Pack for Java**: herramientas básicas para Java.
- **Java**: soporte de lenguaje.
- **Spring Boot Extension Pack** (opcional).
- **Better Comments** (opcional).
- **markdownlint** (opcional, para documentación).
- **Image Preview** (opcional, vista de imágenes en Markdown).
- **Mintlify**: ayuda con documentación automática.

## Notas sobre paquetes en Java

- Un paquete es solo una **carpeta** cuyo nombre coincide con el `package` declarado en el archivo `.java`.
- Si dos clases están en el mismo paquete, puedes usarlas directamente.
- Si están en paquetes diferentes, necesitas `import`.
